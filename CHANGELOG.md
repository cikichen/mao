# 更新日志

#### 3. 动效优化

**地图标记动画增强**:
- 优化标记掉落动画，增加弹跳效果
- 改进标记脉冲动画，更明显的视觉反馈
- 新增标记弹跳动画 `markerBounce`
- 新增足迹轨迹动画 `footprintTrail`
- 优化路径绘制动画，增加透明度变化

**新增现代化动效**:
- 地图缩放动画 `mapZoomIn`
- 标记高亮动画 `markerHighlight`
- 轨迹流动动画 `trajectoryFlow`
- 时间轴指示器动画 `timelineIndicator`
- 侧边栏滑入优化 `sidebarSlideIn`（带模糊效果）
- 卡片翻转效果 `cardFlip`
- 波纹扩散效果 `rippleExpand`
- 渐变背景动画 `gradientShift`
- 文字闪烁效果 `textBlink`
- 骨架屏加载动画 `skeletonLoading`

**性能优化**:
- 所有动画支持 `prefers-reduced-motion` 无障碍访问
- 使用 `will-change` 优化动画性能

#### 4. 锚点功能优化

**AMap 锚点增强**:
- 支持自定义缩放级别
- 支持动画开关和持续时间控制
- 支持标记高亮开关
- 支持中心点偏移（适配侧边栏）
- 标记动画自动停止（3秒后）
- 触发定位完成事件

**Leaflet 锚点增强**:
- 使用 `flyTo` 实现平滑飞行动画
- 支持自定义缩放级别和动画参数
- 支持中心点偏移计算
- 标记图标动态切换（默认/高亮）
- 标记脉冲动画效果
- 触发定位完成事件

**应用层优化**:
- 侧边栏定位：移动端向上偏移100px，桌面端向左偏移150px
- 搜索定位：使用更高的缩放级别(13)和更长的动画时间(1200ms)
- 主应用定位：标准缩放级别(12)和动画时间(600ms)
- 所有定位操作延迟显示详情，优先展示定位动画

**CSS 样式支持**:
- 新增 `.marker-highlight` 类，支持高亮滤镜和动画
- 新增 `.marker-pulse` 类，支持脉冲动画
- Leaflet 标记自定义样式支持

#### 5. 数据加载与缓存优化
- 移除 `index.html` 中对 `window.EVENTS_DATA` 的多文件注入，统一通过 `data/events.json` 异步加载，首屏脚本体积更小
- `DataManager` 引入版本号感知的缓存键（`mao_map_events_v{version}`），自动根据应用版本失效
- 新增缓存有效期控制（默认30天）与过期兜底策略，优先使用本地缓存，网络失败时自动回退到过期缓存或内置 Mock 数据
- `UTILS.storage` 支持移除与元数据存储，便于未来扩展更多缓存策略

#### 6. 开发体验升级（Vite）
- 引入 Vite 作为统一的开发/构建/预览工具，`npm run dev` 取代 Python 静态服务器，默认端口 5173 并支持热更新
- 新增 `npm run build` / `npm run preview` 工作流，`npm run deploy` 自动使用 Vite 生成的 `dist/`
- 通过 `vite-plugin-static-copy` 自动复制 `css/`、`js/`、`images/`、`data/` 与 `config*.js` 等非模块资源，确保构建产物与旧版保持一致

### 📝 API 变更

#### centerToEvent 方法签名更新
```javascript
// 旧版本
centerToEvent(event)

// 新版本
centerToEvent(event, targetZoom = 12, options = {})

// 选项参数
options = {
    animate: true,        // 是否使用动画
    duration: 800,        // 动画持续时间(ms)
    highlight: true,      // 是否高亮标记
    offset: [0, 0]       // 中心点偏移[x, y]
}
```
